# flutter-factory

Комплект для работы с Flutter-приложениями.

Используй его, когда нужно создать, изменить, улучшить или исправить
Flutter-приложение.

У ролей, скиллов и шаблонов в начале файла есть `description`. По нему агент
решает, когда использовать файл.

## Роли

Роли лежат в `roles/<name>.md`.

- `roles/tech-lead.md`
- `roles/flutter-engineer.md`

## Скиллы

Скиллы лежат в `skills/<name>/SKILL.md`.

- `skills/feature-structure/`
